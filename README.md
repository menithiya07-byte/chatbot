1.Rule-Based Chatbot using Python

This repository contains a **rule-based chatbot** created as part of an **internship assignment**.  
The chatbot interacts with users by analyzing their input and responding using predefined rules and pattern matching.

2. Internship Task Overview
Task: Design and implement a basic chatbot that responds to user queries using conditional logic or pattern matching techniques.

3. Project Goals
- Learn the fundamentals of chatbot development  
- Understand how user input can be processed logically  
- Implement a simple conversational flow  
- Explore basic Natural Language Processing (NLP) concepts
